## @polybin/web@0.1.0

- Added support for dark mode in the web interface.
- Introduced a Write/Preview tab interface for creating pastes.
- Enabled GitHub Flavored Markdown rendering with KaTeX math support.
- Integrated a CodeMirror editor for the write tab.

## @polybin/api@0.1.0

- Provided anonymous CRUD APIs for text pastes.
- Documented the API with an OpenAPI schema and provided a Swagger UI.
